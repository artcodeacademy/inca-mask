Inca Mask

In this project we will create a program to design Inca masks using different types of eyes, mouths, noses, and mask shapes. Click in the example to view the final result:

Click here

1) Create a new project and explore your creative side. Draw a new object to be your mask face.

2) Draw at least 2 costumes for the mask face object.

3) Add an object that can be used as a button and write the word "face" on top of it.

4) When you click the button, make it boradcast a message called "change face".

5) When the face object receives the message "change face" make this object change to the next costume.

6) Repeat steps 1 through 5 for the other parts of the mask (eyes, mouth, nose). You are free to draw these parts as you like, experiment with different shapes and colors.

7) Create a button that says "Random", when you press it the program should make a random mask using the different parts.

Hint: Search within the operator blocks to find one that helps you choose the different parts randomly, pay attention to the number of costumes you made for each part of the mask.
